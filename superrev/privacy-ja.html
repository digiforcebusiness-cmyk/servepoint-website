概要

SuperRev はあなたのスマートフォンをリアルタイムのエンジン音シミュレーターに変えます。スマートフォンの GPS 速度とモーションセンサーを読み取って、シミュレートされたエンジン回転数を推定し、運転の仕方に反応する重層的なエンジン音を再生します——すべて端末内で行われます。SuperRev にはアカウントがなく、独自のサーバーも一切運用していません。アプリは Google AdMob が提供するバナー広告を表示します。これらの広告を配信するため、Google の広告 SDK は以下に説明する一定の端末情報と利用情報を収集します。広告を除き、SuperRev は個人データを収集せず、分析やその他の第三者によるトラッキングも一切ありません。

1. 運営者について

SuperRev（「本アプリ」「当社」）は、YOUNESS HARNAFI / DIGIFORCE LLC が公開するモバイルのエンジン音シミュレーターアプリです。本プライバシーポリシーは、Android または iOS で本アプリを使用する際に、アプリが取り扱うデータと取り扱わないデータについて説明します。

2. 位置情報データ

シミュレートされるエンジン回転数は移動速度にも左右されるため、SuperRev は位置情報へのアクセスを要求します。スマートフォンの GPS 速度はアプリの実行中に継続的に読み取られ、リアルタイムで、あなたが聞くエンジン音の計算に使用されます。

  • エンジンシミュレーションに使用される位置情報は、完全にあなたの端末内で処理されます。 オペレーティングシステムの位置情報サービスによって読み取られ、速度と回転数の計算に即座に使用された後、破棄されます。履歴としてディスクに書き込まれることは決してなく、SuperRev がそれを当社に送信したり、いかなるサーバーにも保存したりすることはありません。
  • これとは別に、Google AdMob 広告 SDK（第6条）は、関連性のある広告の選択を助けるため、たとえば IP アドレスからおおよその位置を推定する場合があります。この広告関連の処理は SuperRev ではなく Google が行います。

位置情報の許可を拒否または取り消しても、アプリは引き続き動作します——その場合、エンジンシミュレーションはモーションセンサーとアプリ内のスロットル操作に依存します。

3. モーションおよびその他のセンサー

SuperRev は端末の加速度センサー（および関連するモーションセンサー）を読み取り、加速、ブレーキ、スマートフォンの向きを検出します。これにより、アプリは「スロットル」を感知し——たとえばスマートフォンを前方に振るとシミュレートされたエンジンが吹け上がります——車両がどちらを向いているかを較正できます。

センサーの読み取り値はリアルタイムでエンジン音の計算に使用され、即座に破棄されます。記録、保存、送信はいずれも行われません。

4. エンジン音声

SuperRev はアプリに組み込まれたオーディオエンジンを通じてエンジン音を再生します。音声はアプリ内に同梱された音声ファイルから完全に端末内でレンダリングされ、スマートフォンのスピーカー、またはあなたが選んだ接続済みの Bluetooth／有線出力を通じて再生されます。オーディオエンジンは個人データを処理せず、ネットワーク接続も一切行いません。

5. 端末内にのみ保存されるデータ

アプリは、セッション間で設定が保持されるよう、少量の情報をローカルに保存します。そのいずれも端末外へ送信されることはありません。

  • 選択中の車 — 現在有効なエンジンパック
  • 較正 — スマートフォンの取り付け方に合わせてモーション検知が一致するよう設定する前方軸の向き
  • 環境設定 — アプリの設定とスイッチ

これらのデータは端末のローカルアプリストレージに保存されます。アプリをアンインストールすれば、いつでも消去できます。

6. 広告 — Google AdMob

SuperRev は Google のモバイル広告サービスである Google AdMob を使用してバナー広告を表示します。広告の選択、配信、計測、表示頻度の制限のため、アプリに組み込まれた Google Mobile Ads SDK は一定の情報を収集し Google に送信します。これには通常、以下が含まれます。

  • 広告識別子（Android 広告 ID または iOS の IDFA）— リセット可能な端末ごとの広告用 ID
  • 端末情報 — 端末モデル、OS のバージョン、言語、画面の特性など
  • おおよその位置 — 通常は IP アドレスから導かれます（GPS 座標ではありません）
  • 広告操作データ — 広告が表示・閲覧・タップされたかどうか

この情報は独立した第三者として行動する Google によって収集・使用され、Google 自身のポリシーに従います——Google プライバシーポリシー、およびGoogle のサービスを利用するサイトやアプリからの情報の Google による使用についてをご覧ください。SuperRev 自体は、この処理を通じてあなたの広告識別子、連絡先情報、正確な位置を受け取ることはありません。

広告のパーソナライズはいつでも制限できます。Android では設定 → プライバシー → 広告（広告 ID の削除またはリセットも可能）から、iOS では設定 → プライバシーとセキュリティ → トラッキングおよびApple 広告から行えます。広告 ID のリセットや削除、またはパーソナライズの無効化を行っても広告は消えませんが、パーソナライズされない広告になります。

AdMob 以外に、SuperRev は分析 SDK、計測・アトリビューション SDK、その他の第三者トラッキング技術を一切含みません。

7. アプリが要求する権限

アプリは提供する機能に必要な権限のみを要求します。

  • 位置情報（正確およびおおよそ） — 回転数シミュレーションのため GPS 速度を読み取る（第2条）
  • 身体／モーションセンサーおよび高サンプリングレートセンサー — スロットルと向きの検知のため加速度センサーを読み取る（第3条）
  • インターネット — Google AdMob SDK が広告を読み込むために使用（第6条）
  • 広告 ID — Google Mobile Ads SDK が広告を配信できるよう宣言（第6条）

アプリは連絡先、カレンダー、マイク、カメラ、写真、SMS へのアクセスを要求せず、エンジンシミュレーションはアプリがフォアグラウンドにあるときのみ動作します。

8. 当社が収集しないデータ

SuperRev 自体は、ユーザーデータを保存するサーバーを一切運用していません。Google AdMob が取り扱う広告データ（第6条）を除き、当社は以下を収集・保存・送信しません。

  • 氏名、メールアドレス、電話番号（アプリにサインアップやアカウントはありません）
  • あなたの GPS 座標、ルート、位置情報の履歴
  • モーションセンサーの記録や運転履歴
  • 当社が収集するアプリ利用分析、テレメトリ、クラッシュレポート
  • 連絡先、写真、カレンダー、マイク、カメラのデータ

9. 子どものプライバシー

SuperRev は 13 歳未満の子どもを対象としておらず、13 歳以上の一般の利用者向けです。アプリは子どもから個人情報を意図的に収集することはありません。アプリは広告を表示するため、Google Play の「ファミリー向け」プログラムには登録されていません。ご質問のある保護者の方は、下記のメールアドレスまでご連絡ください。

10. あなたの権利

SuperRev はあなたに関する個人情報をいかなるサーバーにも保持していないため、当社が遠隔でアクセス・訂正・削除するものはありません。あなたは以下を完全に管理できます。

  • 端末内のデータ — アプリをアンインストールすると、ローカルに保存されたすべてが消去されます
  • 位置情報とセンサーへのアクセス — Android の設定 → アプリ → SuperRev → 権限、または iOS の設定 → SuperRev で切り替えます
  • 広告 — 端末のプライバシー設定で広告 ID をリセットまたは削除し、広告のパーソナライズを無効にします（第6条）

Google が保持する広告データについては、Google プライバシーポリシーに基づき Google に直接権利を行使できます。欧州経済領域、英国、カリフォルニア州、その他の適用されるプライバシー法を有する法域の利用者も、あらゆる請求について当社にご連絡いただけます。当社自身は個人データを保持していないため、そうした請求の多くは、あなたに関するデータが当社に存在しないことを確認する回答となります。

11. データの保持

SuperRev は自社サーバーにデータを一切収集しないため、当社はユーザーデータを保持しません。位置情報とセンサーの読み取り値は、エンジン音を計算する間、端末メモリ内に一時的に存在するだけです。端末内の環境設定は、アプリがインストールされている限りにおいてのみ保持されます。Google AdMob が収集する広告データは、Google 自身の保持ポリシーに従って Google が保持します。

12. 国際的な利用者

SuperRev は世界中で利用できます。アプリ自体は個人データを国境を越えて転送しません。Google AdMob SDK が行う広告リクエストは Google のグローバルインフラから配信される場合があり、Google 自身のプライバシー慣行の対象となります。

13. 本ポリシーの変更

本プライバシーポリシーを変更する場合、更新版は上部に新しい「最終更新日」を付して、この URL に掲載されます。データの取り扱いに影響する重要な変更は強調して示されます。変更後もアプリの利用を継続することは、改定後のポリシーへの同意を意味します。
